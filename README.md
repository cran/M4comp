# Installation

You can install the **development** version from [Github](https://github.com/bsouhaib/M4comp)

```s
# install.packages("devtools")
library(devtools)
install_github("bsouhaib/M4comp") 
```

The **stable** version on CRAN is coming soon.

# Usage

```s
library(M4comp)

print(M4)
plot(M4[[1]])
print(M4[[1]])

```
